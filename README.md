Aplicativo mobile de formulário de cadastro desenvolvido em React Native com Expo para a disciplina Mobile Application Development 

---

## 📋 Sobre

O app permite que o usuário preencha um formulário com suas informações pessoais e visualize os dados cadastrados na própria tela após o envio.

---

## ⚙️ Funcionalidades

- Formulário com campos de **Nome, Curso, Disciplina e Descrição**
- Exibição dos dados preenchidos após envio
- Controle de estado com `useState`
- Execução de ação na inicialização com `useEffect`

---

## 🛠️ Tecnologias

- React Native
- Expo
- @expo/vector-icons
- react-native-safe-area-context
